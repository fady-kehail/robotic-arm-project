3D Printed Robotic Arm

Graduation project developed during my Mechatronics Engineering degree.

This project involved designing and building a robotic arm with a 3D-printed robotic hand, capable of controlled finger and arm movements using servo motors.

Key Features

3D printed robotic hand structure

Servo motor actuation for fingers and arm joints

Embedded control system

Integrated electronics and actuator control

Technologies

Arduino
C/C++
Servo Motors
3D Printing
